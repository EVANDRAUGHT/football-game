🎨 视频选择器样式演示

未选中状态（橙色警告）

当用户还没有选择视频时，选择框会显示橙色背景和边框，文字为斜体，以提醒用户需要选择。

设计理念: 使用橙色作为警告色，配合斜体文字和警告图标 ⚠️，形成强烈的视觉提示，引导用户完成选择操作。

已选中状态（绿色确认）

当用户选择了视频后，选择框会变为绿色边框，背景色降低透明度，表示已成功选择。

设计理念: 使用绿色表示确认状态，恢复正常字体，视觉上更加低调但依然保持辨识度。

🖱️ 交互式状态切换

点击下方按钮体验状态切换效果，观察颜色、边框和字体的变化。

提示: 将鼠标悬停在选择框上，观察发光效果和边框颜色变化。

✨ 悬停效果演示

将鼠标移到下方任意选择框上，观察边框发光和阴影效果。

动画效果:
  • 边框颜色过渡: 0.3秒
  • 发光阴影: 0 0 15px rgba(0, 255, 136, 0.3)
  • 平滑过渡: transition: all 0.3s ease

🔧 技术实现

核心 CSS 代码：

/* 未选中状态 */
.not-selected {
    background: rgba(255, 159, 64, 0.15);
    border: 2px solid rgba(255, 159, 64, 0.5);
    color: rgba(255, 159, 64, 0.9);
    font-style: italic;
}

/* 已选中状态 */
.selected {
    background: rgba(0, 255, 136, 0.1);
    border: 2px solid rgba(0, 255, 136, 0.4);
    color: var(--text-primary);
}

/* 悬停效果 */
select:hover {
    border-color: var(--primary);
    box-shadow: 0 0 15px rgba(0, 255, 136, 0.3);
}

JavaScript 状态切换：

select.onchange = (e) => {
    if (e.target.value) {
        select.className = 'selected';
    } else {
        select.className = 'not-selected';
    }
};
