使用指南

GoalScope · 足球视频智能分析系统

1
上传足球视频

点击左侧首页，在上传区拖拽或点击上传视频（支持 MP4、MOV 格式）。

2
配置并开始分析

勾选"生成精彩视频"后点击开始分析，系统将自动识别球员并生成能力画像。

3
个人中心 / 历史记录

在左侧导航点击个人中心，可查看所有历史分析记录，支持搜索、排序和下载精彩视频。

4
AI 智能问答

点击左侧AI 智能问答，用自然语言提问，例如"哪名球员防守最强？"，历史对话自动保存。

常见问题
Q: 分析进度卡住？ → 等待 3 分钟，首次运行需下载YOLO模型
Q: 视频无法播放？ → 请使用 Chrome 或 Edge 最新版浏览器
Q: AI 助手无响应？ → 未配置 DeepSeek API Key 时为演示模式
